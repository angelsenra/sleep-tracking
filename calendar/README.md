# CalendarCreator
Create useful calendars.

### Dependencies
- Python 3
- pygame
- [Dejavu Fonts](https://dejavu-fonts.github.io/) (You might already have them installed)

### Usage

To get a custom calendar first configure `config.json` and then run `main.py`.
